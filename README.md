# @WINT365 - 블랙 테마 로그인 웹사이트

기존 WPF 로그인 애플리케이션을 웹으로 포팅한 블랙 테마의 미니멀한 로그인 시스템입니다.

## 🎨 특징

- **블랙 테마**: 세련된 다크 모드 디자인
- **미니멀 UI**: 깔끔하고 현대적인 인터페이스
- **타이핑 애니메이션**: @WINT365, SIGN UP, CODE INJECTION 텍스트 애니메이션
- **반응형 디자인**: 모바일과 데스크톱 모두 지원
- **드래그 가능**: 창을 드래그하여 이동 가능

## 🚀 기능

### 로그인 모드
- 이메일과 비밀번호로 로그인
- Remember me 기능
- 입력 검증 및 에러 메시지

### 회원가입 모드
- 새 계정 생성
- 비밀번호 확인
- 이메일 중복 검사

### 코드 등록 모드
- 액세스 코드 등록
- 사용자 인증

## 🎯 주요 기능

### 애니메이션
- **@WINT365**: 메인 타이틀 타이핑 애니메이션
- **SIGN UP**: 회원가입 모드 전환 시 애니메이션
- **CODE INJECTION**: 코드 등록 모드 전환 시 애니메이션
- **깜빡임 효과**: 애니메이션 완료 후 부드러운 깜빡임

### 모드 전환
- **SIGN UP** 클릭: 회원가입 모드로 전환
- **CODE** 클릭: 코드 등록 모드로 전환
- **CANCEL** 클릭: 이전 모드로 복원

### 입력 필드
- **플레이스홀더**: 포커스 시 자동 숨김/표시
- **아이콘**: 이메일(✉), 비밀번호(🔒), 코드(🔑), 등록(📝)
- **검증**: 실시간 입력 검증

## 🎨 디자인 특징

- **색상**: 블랙(#1A1A1A), 화이트(#FFFFFF), 그레이(#CCCCCC)
- **폰트**: Segoe UI
- **투명도**: 반투명 배경 (opacity: 0.9)
- **그림자**: 부드러운 드롭 섀도우 효과
- **테두리**: 둥근 모서리 (border-radius: 6px)

## 📱 반응형 지원

- **데스크톱**: 400px 고정 너비
- **태블릿**: 90% 너비 (최대 400px)
- **모바일**: 95% 너비, 폰트 크기 조정

## 🔧 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: Flexbox, Grid, 애니메이션
- **JavaScript ES6+**: 모듈화된 코드 구조
- **LocalStorage**: 사용자 설정 저장
